# LittlefootCMS

The Big CMS with a little footprint.

Visit http://littlefootcms.com/manual for documentation.

## Issues

The preferred convention for issue titles is as follows:

`<affected area> - <issue>`

> ie, lf admin - add 'anonymous' to ACL tool
